# Browser Automation

This folder contains a small Codex + Playwright CLI automation setup.

## Commands

```powershell
npm install
npm run browser:install
npm run browser:visit -- https://example.com artifacts/example.png
npm test
```

The reusable Codex skill lives in `skills/playwright-cli-browser/`. A local installed copy is also available at `%USERPROFILE%\.codex\skills\playwright-cli-browser`.
